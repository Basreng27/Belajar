Variable

Variable dibagi menjadi 3 :
1. var
2. let
3. const

============================================================================================================================================

Let

Variabel yang didefinisikan dengan let tidak dapat dideklarasikan ulang.
Variabel yang didefinisikan dengan let harus dideklarasikan sebelum digunakan.
Variabel didefinisikan dengan membiarkan memiliki Lingkup Blok.
Variabel yang dideklarasikan di dalam blok {} tidak dapat diakses dari luar blok.
let x = 10;
Disini x adalah 10
{
let x = 2;
// Disini x adalah 2
}
// Disini x adalah 10

============================================================================================================================================

Const

Variabel yang didefinisikan dengan const tidak dapat dideklarasikan ulang.
Variabel yang didefinisikan dengan const tidak dapat ditugaskan kembali.
Variabel yang didefinisikan dengan const memiliki Lingkup Blok.

penulisan yang benar

const PI = 3.14159265359;

penulisan yang salah

const PI;
PI = 3.14159265359;

Kapan Saat menggunakan const?

membuat Array (jika isinya konstant tidak berubah ubah)
membuat Object (jika isinya konstant tidak berubah ubah)
membuat Function (jika isinya konstant tidak berubah ubah)
membuat RegExp (jika isinya konstant tidak berubah ubah)
